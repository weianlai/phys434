# Part A
#
# The rate of cosmic ray background I have chosen is 3, which is plugged right into the
# poisson function. With increasing time of observation, the poisson parameter increases,
# since the variable is the product of the rate and the time interval. This poisson parameter,
# or lambda, as it gets larger, will peak later at a smaller probability. This behavior makes
# the curve more flattened as time interval is increased.

xaxis = np.arange(25);
rate = 3;

for x in range(1,5):
    dist = poisson(x * rate)
    plt.plot(xaxis, dist.pmf(xaxis), label = '%i days' %x);

plt.title('Poisson Distribution')
plt.xlabel('Days');
plt.ylabel('Probability');
plt.legend();
plt.show();

# Part B
#
# The sum of the distribution should retain its Poisson behavior because the events are still
# occuring independently without effecting the next events. This follows the Central Limit Theorem,
# because as we put more days in, we increase the probability of occurance in later days, lowering
# the existing peak. As a result, the curve becomes more symmetrical at the mean, slowly becoming
# more like a normal distribution.

def poissonEq(mu, k):
    return np.exp(-mu) * mu**k / np.math.factorial(k)

for x in range(1,7):
    plt.scatter(x,poissonEq(rate,x))

plt.title('Poisson Distribution')
plt.xlabel('Days');
plt.ylabel('Probability');
plt.show();

# As we average more days, we can see that the curve increases, eventually getting very close to one
# when the days becomes very large. The behavior of the curve can be visualized by looking at the area
# of the original poisson ditribution curve. Most of the area is near the mean, or value of lambda,
# so the probability density adds up very quickly. At the skew, or the tail, there isn't much area, so
# there isn't much to add to the probability density.
